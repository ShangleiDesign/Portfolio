RememberStuff - An Elderly Caretaking Solution

Project Info

RememberStuff is a web application embedded in a standalone device that support the elderly and caregivers. It seeks to provide meaningful solution for caregivers of independently minded individuals who experience early cognitive decline.

What I did

  • UX & UI Design
  • Graphic & Icon Design
  • Usability Studies
  • Front-end Development

Project Background

I am fortunate to be the lead designer on this project that provides care and restore dignity and independence for a loved one with dementia.

This is a long term project that crosses several iterations and involves multiple rounds’ design changes. In this project, I went through the processes of market research, user study, low-fid and high-fid prototypes, graphic and icon design, and supported part of the front-end development (caregiver portal).

One greatest take-away from this project was that an ideal design for the public could be a worst design to a particular user group. The main user group is elderly people with cognitive declines – visual, hearing and memory. The UI style, font type and size design are catered to the particular user group - accessibility is a big topic for this particular project.

End User Side Design Artifacts

At the end user side, the device is a standalone touch screen that requires minimal setup. Once setup the system will be up and running the end user can just use the touch screen to interacte with the large monitor.

At the end user side, there are four sections:

  1. Calendar
  2. Messages
  3. Remember
  4. Activities

RememberStuff Touch Screen Introduction

Calendar Page

Calendar

Calendar shows the activities or appointments set by the caregiver to help the end user organize the daily routine, which could be family activities, medicine reminder and etc. On the calendar, the current date will automatically be selected and will be indicated by a red dot. Today's agenda will also display by default on the right side of the screen.

Messages Page

Messages

The messages page is the centualized location for communication, where the end user can chat with their caregivers or the loved ones via messages or video calls.

Remember Page

Remember

The remember page serve as a notebook for storing all kinds of topics that comes with detailed desciptions. Those topics could vary from planning a trip with friends, a doctor appointment, a food food recipe, or even some family photo memories. Those remember items are created and maintained by the end-users caregivers.

Activities Page

Activities

At the activities page, users can play puzzles and take quizzes with personalized content tailered to what they need to help rememberit.

Caregiver Portal

The caregiver portal is the CMS for the caregivers to populate and manage the content for the end user side, and also it is the location where they can communicate directly with their cared ones.
